About / Data sources

This is a static, GitHub Pages friendly Leaflet template designed for a wildfire web map portfolio.

How to use

  1. Replace placeholder GeoJSON files in data/ with your processed outputs (GeoJSON/TopoJSON).
  2. Edit js/config.js to add menus, datasets, legends, and popup fields.
  3. Deploy on GitHub Pages (Settings → Pages).

Suggested real datasets (BC)

  • BC Wildfire Service incident perimeters / points (as available)
  • Fire management boundaries
  • Climate normals / precipitation / wind (aggregate to raster tiles or vector summaries)
